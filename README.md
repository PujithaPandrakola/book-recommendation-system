# Book Recommendation System

This project recommends books based on user reading history and content similarity using TF-IDF and cosine similarity.

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn

## Features
- User-based book recommendations
- TF-IDF vectorization
- Cosine similarity matching
- Interactive Streamlit UI

## Run Project

```bash
pip install -r requirements.txt
streamlit run app.py
