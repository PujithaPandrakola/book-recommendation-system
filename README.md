# 📚 Book Recommendation System

A Machine Learning based Book Recommendation System that recommends books to users based on reading history and content similarity using **TF-IDF Vectorization** and **Cosine Similarity**.

---

## 🚀 Live Demo

🔗 https://book-recommendation-system-nc8pqit5hjajpreughgbu8.streamlit.app/

---

## 📌 Project Overview

This project recommends books to users by analyzing:

- User reading history
- Book title
- Author
- Publisher information

The recommendation engine uses **Natural Language Processing (NLP)** techniques to find similar books and provide personalized suggestions.

---

## ✨ Features

✅ User-based book recommendations  
✅ Content-based filtering  
✅ TF-IDF text vectorization  
✅ Cosine similarity matching  
✅ Interactive Streamlit web interface  
✅ Fast and responsive recommendation system
